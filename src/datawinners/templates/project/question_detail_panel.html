• What is this?
    Each question in a questionnaire is identified by its own unique Question Code. This Question Code tells DataWinners what question is answered by the Data Sender.

    What do I need to do?
    Enter a 1 or 2 character code (letters, numbers or both). Your Data Senders type the "+" code followed by the question code (example: +rpd) for each question. This will tell DataWinners what question the Data Sender is answering.

  • What is this?
    Questionnaires collect data in several formats: numbers, text, multiple choice and dates. The Answer Type identifies which format the Data Sender must use when answering the question.

    What do I need to do?
    Select the Answer Type that the Data Sender will use when answering the question. Answers can be numbers, text, multiple choice, or dates.
    When using the Multiple Choice option, Data Senders will type in the letter (A, B, C etc.) that corresponds to the correct answer NOT the text in the answer field.
