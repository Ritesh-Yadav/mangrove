Step 2: Subjects

What is this?
Your Data Senders collect data about Subjects, which are often people (patients, farmers) or things (health clinics, retail shops). Subjects can also be staff who send you their monthly activity reports.

DataWinners gives you the option to Register Subjects by entering basic data (name, location, subject type) about each subject. DataWinners will then give each subject a Unique Identification Number ('ID'). Data Senders will always identify Subjects by their ID. This saves time and space in the SMS because Data Senders will not have to type the name, location and other information about the subject each time they send data.

What do I need to do?
Decide what information you want to use to identify and register each Subject ( For example: first name, last name, location, mobile phone number, etc.). You then have three ways to register this information: 1) Manually enter the details using the online form, 2) import a list ; or, 3) Print out the Subject Registration Questionnaire, give it to your Data Senders and ask your Data Senders to register Subjects themselves. Import a List of Subjects

What is this?
Add a list of subjects all at once by importing an Excel (.xls) or a Comma Separated Value (.csv) file

What do I need to do?
Follow these five easy steps to import a list of Subjects:
1) Open a new Excel spreadsheet. In cell A1, type form_code. In cell B1, type T. In cell C1 type N. In cell D1 type M. In cell E1 type S. In cell F1 type L. In cell G1 type G. In cell H1 type D.
2) In cell A2, type REG. In cell B2, type the Subject type (example: Clinic). In cell C2, type the name of your Subject (example: Northern Clinic). In cell D2, type the mobile telephone number of the Subject. In cell E2, type a unique ID code for the Subject. DataWinners will automatically generate a unique ID number if you leave this cell blank. In cell F2, type the location of the Subject. In cell G2 type the GPS coordinates of the Subject (Right click on cell G2, choose "Format Cells", Choose "Text", then click "OK". Then enter the GPS coordinates in the following format: 16.8665 47.53150. Remember do not include a comma). In cell F2 type a short description of the Subject.
3. Repeat step #2 for each Subject starting in cell A3. Every Subject thereafter will occupy a new row (A4, A5, A6 etc.)
4. Save the file as (.csv) or (.xls)
5. Click on blue "Import List" button and follow instructions
