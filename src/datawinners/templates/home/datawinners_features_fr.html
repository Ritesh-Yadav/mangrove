DataWinners dispose des fonctionnalités suivantes pour vous aider à collecter des données émanant du terrain

Editeur de Questionnaire

  • Transformez votre liste de questions clés en un questionnaire en ligne
  • Spécifiez le type de réponse : Texte, chiffre, date, choix multiple, etc.
  • Fixez des critères de validation pour améliorer la précision. Exemple : La réponse doit se situer entre 1 et 100
  • Surveillez le nombre de caractères des SMS requis pour envoyer les réponses aux questions clés

Phase de Test

  • Testez au cours de la phase de test : Finalisez votre questionnaire et formez vos Expéditeurs de Données
  • Envoyez les réponses tests via :
    • SMS en utilisant votre téléphone
    • Notre émulateur téléphonique en ligne
    • Formulaire Web
  • Dès que vous avez activé votre projet, commencez à collecter des données réelles en utilisant une base de données vide

Des Canaux Multiples: SMS, Internet

  • Utilisez soit l’Internet, les SMS, ou les deux pour collecter vos données
  • Vos expéditeurs de données (personnels de terrain) peuvent utiliser leur téléphone individuel. Même les plus simples des téléphones peuvent envoyer des données par SMS
  • Collectez les données en commençant par les régions les plus éloignées

Rappels & Messages

  • Créez des rappels automatiques pour aviser les Expéditeurs de Données qui n’ont pas encore envoyé leurs réponses
  • Envoyez des accusés de réception automatiques et des messages d’erreur à vos Expéditeurs de Données ; exemple : « Félicitations Shilpa ! Nous avons bien reçu vos données »

Cartographie

  • Visualisez les points de collecte de données affichés avec précision sur une carte
  • Triez la carte selon la période, la localisation géographique, les questions du rapport, ainsi de suite !

Analysez les Données

  • Analysez les données avec l’aide des listes et des cartes
  • Effectuez de simples calculs statistiques sur vos données
  • Exportez vos données en un seul clic
  • Triez les données pour les consulter par date, localisation ou question
