WHAT IS DATAWINNERS

  • Features
  • How it works
  • Benefits

Create a Project:

  1. Each DataWinners Project = 1 Questionnaire
  2. Name your project for easy identification
  3. Set the questionnaire code so you can collect answers to multiple questionnaires simultaneously using one central telephone number
  4. Set how often you want to receive your answers
  5. Set the subject of your questionnaire

Build a Questionnaire

  1. Turn your list of key questions into an online questionnaire
  2. Set the answer type for each question
  3. Set criteria for each answer to improve accuracy
  4. Monitor the SMS character count required to send in answers to key questions
  5. Review the instructions for each question

Decide Between Test and Active States

  1. DataWinners has two project states: Test and Active
  2. Finalize your questionnaire and train your Data Senders in Test State
  3. Activate your project to wipe clean your data base of all test answers sent in during training exercises
  4. Start collecting real data with an empty database once you activate your project.

Test the questionnaire

  1. Print out your draft questionnaire and fill in test answers by hand
  2. Send in test answers via SMS using your phone
  3. Send in test answers using the online telephone emulator
  4. Send in test answers using your questionnaire via web form

Decide who can send in answers

  1. Choose between:
    1. Everyone
    2. Only Data Senders you authorize
  2. Tell DataWinners the name, location and telephone number of each Data Sender authorized to send in answers
  3. Register Data Senders one-by-one using the web form
  4. Import a list of authorized Data Senders

Register the subjects of your questionnaire

  1. If you are collecting data about individual subjects such as people (medical records), places (number of households in a commune), things (health center stock report) or events (participation at a training), register each subject first for two reasons:
    1. See your subjects displayed on DataWinners map
    2. Set a unique identification number for each subject to improve data reporting accuracy
  2. Tell DataWinners the name, location and telephone number of each Subject
  3. Register Subjects one-by-one using the web form
  4. Import a list of Subjects
  5. Register Subjects via SMS

Set Deadlines and Reminders

  1. Set a deadline for your Data Senders to send in their answers
  2. Send reminders to Data Senders who have not yet sent in their answers

Modify your Questionnaire

  1. Delete, add and edit your questionnaire later on as your needs change
  2. Change answer type and criteria for any question

See your Data

  1. See your Subjects plotted on a map
  2. Review a spreadsheet of all Data Sender submissions
  3. Filter to see only successful, or unsuccessful, or deleted submissions
  4. Analyze your data in a spreadsheet filtered by date and subject
  5. Perform simple statistical calculations on your data
  6. Export your data in a single click
