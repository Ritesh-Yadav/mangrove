DataWinners provides the following features to help you collect data from the field.

Questionnaire Builder

  • Turn your list of key questions into an online questionnaire
  • Specify the answer type: Text, number, date, multiple choice, etc.
  • Set validation criteria to improve accuracy. Example: Answer must be between 1 - 100
  • Monitor the SMS character count required to send in answers to key questions

Test Phase

  • Experiment in the test phase: Finalize your questionnaire and train your Data Senders
  • Send in test answers via:
    • SMS using your phone
    • Our online telephone emulator
    • Internet
  • Start collecting real data with an empty database once you activate your project

Multiple Channels: SMS, Internet

  • Use either the Internet, SMS or both to collect your data
  • Your Data Senders (field staff) can use their personal phone. Even the simplest phones can send data using SMS
  • Collect data from the most remote areas

Reminders & Messages

  • Send automatic reminders to your Data Senders
  • Send your Data Senders automatic success and error messages, e.g. “Congrats Shilpa! We successfully received your data”

Mapping

  • See all data collection points accurately displayed on a map
  • Filter map based on time period, geographic area, report questions and more

Data Analysis

  • Analyze data using spreadsheets and maps
  • Perform simple statistical calculations
  • Export your data in a single click
  • Filter to view data by date, location, or question
